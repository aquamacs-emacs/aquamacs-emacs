1.4.3 Searching

As in Vi, searching is done by / and ?. The string will be searched literally by default. To invoke a regular expression search, first execute the search command / (or ?) with empty search string. (I.e, type / followed by <RET>.) A search for empty string will toggle the search mode between vanilla search and regular expression search. You cannot give an offset to the search string. (It is a limitation.) By default, search will wrap around the buffer as in Vi. You can change this by rebinding the variable vip-search-wrap-around. See Customization, for how to do this.
