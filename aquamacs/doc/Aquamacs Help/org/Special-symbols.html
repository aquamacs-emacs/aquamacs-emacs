11.6.1 Special symbols

You can use LaTeX macros to insert special symbols like ‘\alpha’ to indicate the Greek letter, or ‘\to’ to indicate an arrow. Completion for these macros is available, just type ‘\’ and maybe a few letters, and press M-<TAB> to see possible completions. Unlike LaTeX code, Org mode allows these macros to be present without surrounding math delimiters, for example: 

     Angles are written as Greek letters \alpha, \beta and \gamma.

During export, these symbols will be transformed into the native format of the exporter backend. Strings like \alpha will be exported as &alpha; in the HTML output, and as $\alpha$ in the LaTeX output. Similarly, \nbsp will become &nbsp; in HTML and ~ in LaTeX. If you need such a symbol inside a word, terminate it like this: ‘\Aacute{}stor’.

A large number of entities is provided, with names taken from both HTML and LaTeX, see the variable org-html-entities for the complete list. ‘\-’ is treated as a shy hyphen, and ‘--’, ‘---’, and ‘...’ are all converted into special commands creating hyphens of different lengths or a compact set of dots.
