4.1 Bringing some Emacs 24 behavior back

Expert users who find themselves confused by some changes to the Emacs behavior that they are used to, either due to the upgrade to Emacs 24, or to Aquamacs, may undo configuration changes quickly by using the Emacs Lisp code shown at http://www.emacswiki.org/emacs/AquamacsEmacsCompatibilitySettings in their Preferences.el file.
