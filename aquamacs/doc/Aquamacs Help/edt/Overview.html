1 Overview of the EDT Package

This manual describes version 4.0 of the EDT Emulation for Emacs 19 and above. It comes with special functions which replicate nearly all of EDT's keypad mode behavior. It sets up default keypad and function key bindings which closely match those found in EDT. Support is provided so that users may reconfigure most keypad and function key bindings to their own liking.

Version 4.0 contains several enhancements (see Changes).
