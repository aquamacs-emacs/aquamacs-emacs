Keyboard Shortcuts

Emacs has a well-defined set of keyboard shortcuts, which Aquamacs revises to accomodate OS X conventions. See Table 2 and Table 3 for details.


Table 2: Aqua-specific keyboard shortcuts implemented in Aquamacs. This assumes that the ``Option Key is Meta'' (Options menu).
Shortcut Elisp Command Function
Apple-N new-frame-with Create new buffer
  -new-scratch  
Apple-O mac-find-file- Open a file
  other-frame  
Apple-Shift-S mac-save-file-as Save as
Apple-A mark-whole-buffer Select all text
Apple-V yank Paste text
Apple-C clipboard-kill-ring-save Copy text
Apple-Option-C aquamacs-clipboard-kill- Copy text from secondary selection
  ring-save-secondary  
Apple-X clipboard-kill-region Cut text
Apple-Option-X aquamacs-clipboard- Cut text from secondary selection
  kill-secondary  
Apple-S save-buffer Save file
Apple-L goto-line Go to specified line
Apple-F isearch-forward Search
Apple-G isearch-repeat-forward Repeat search
Apple-E aquamacs-use-selection-for-find Use selected text for next search
Apple-W intelligent-close Close selected window
Apple-M iconify-or-deiconify- Minimize window
  frame to the Dock
Apple-. keyboard-quit) Keyboard quit
Apple-, customize Show Customization Buffer
Apple-; toggle-mac-option- Change Option key function
  modifier  
Apple- / / ' (un)comment-region-or-line Comment out or in the
  current line or region if marked  
Apple-Backspace kill-whole-visual-line Deletes the current line
Apple-Delete kill-visual-line Deletes the remainder of the
    current line
Apple-Q save-buffers-kill-emacs Save file, exit program
Apple-Z undo Undo
Apple-Shift-Z redo Redo
Mouse-drag   select text
Option-Mouse-drag   select text as secondary selection
